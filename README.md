# insomnia-template-api-calls
Repository to track changes &amp; version control for Insomnia UNTP API Calls using built-in Insomnia Git Integration tool.


## Instructions
To clone the repository on Insomnia:
1. Create a new project, name the project accordingly. Select the project type as "Git Sync" 
2. Choose repository = "pyx-industries/insomnia-template-api-calls", branch = "master"
3. Clone the repository.

## Development Lifecycle
1. Currently, we are using the master branch to keep track of changes.
2. Before making any updates, ensure you've git pull the repository to the latest version.
3. Any changes will be detected by the Insomnia tool on the bottom left corner.
4. Once you are happy with the changes, commit & push to the remote repository.
